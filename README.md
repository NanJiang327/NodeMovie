# Node Blog (express 4.0)
> A movie site based on node.js + express 4.0 + ejs + mongoDB

# Build Setup
``` bash
# install dependencies
npm install

#server with hot reload at localhost:3000
nodemon app.js
```

# Tech stack
- Node.js
- Express 4
- ejs
- MongoDB
- Mongoose
- Marked (Markdown to HTML parser)
- jQuery
- Bootstrap 3
- mocha
